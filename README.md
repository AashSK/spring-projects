# Spring Boot Projects Portfolio

This repository is a collection of my Spring Boot projects. This repository is a central index of all the Spring Boot applications I've developed. These projects cover a range of topics, from beginner-friendly CRUD applications to more advanced concepts such as Spring Security, REST APIs, JPA, microservices, and Docker.

## Projects

- [Project 1](https://github.com/AashSK/Event-Ticket-Microservice) – RESTful API for a Event Ticketing system (Spring Boot, Kafka, Keycloak, MySQL, Flyway Migration, Spring API Gateway).
- [Project 2](https://github.com/AashSK/spring-videostreaming-microservices) – Microservices-Based RESTful API for a Video Streaming Platform (Spring Boot, Eureka, Spring Cloud, MySQL).
- [Project 3](https://github.com/AashSK/spring-boot-taskmanager) – RESTful API for a task management system with Spring Boot bestpractices.
